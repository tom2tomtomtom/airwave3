# AIrWAVE Platform - User Guide

This guide provides instructions for using the AIrWAVE platform, a comprehensive digital ad execution platform that helps businesses produce digital assets at scale.

## Getting Started

### Logging In

1. Navigate to the AIrWAVE platform login page
2. Enter your email and password
3. Click "Sign In"

If you don't have an account, click "Sign Up" to create a new account.

### Dashboard Overview

The dashboard provides an overview of your recent activity and quick access to all platform features:

- Client selection
- Asset management
- Template management
- Content generation
- Visual matrix
- Approval workflow
- Export management

## Client Management

### Selecting a Client

1. Click the client selector in the top navigation bar
2. Choose a client from the dropdown menu
3. The selected client will be used for all content creation activities

### Creating a New Client

1. Click the client selector in the top navigation bar
2. Click "Create New Client"
3. Fill in the client details:
   - Name
   - Description
   - Logo (optional)
   - Primary color
   - Secondary color
4. Click "Create Client"

## Asset Management

### Uploading Assets

1. Navigate to the Assets page
2. Click "Upload Assets"
3. Drag and drop files into the upload area or click to browse files
4. Add tags to your assets for easy organization
5. Click "Upload" to complete the process

### Managing Assets

1. Use the search bar to find specific assets
2. Filter assets by type using the dropdown menu
3. Click on an asset to view details
4. Use the edit button to modify asset tags
5. Use the delete button to remove assets

## Template Management

### Importing Templates

1. Navigate to the Templates page
2. Click "Import Template"
3. Enter the Creatomate template ID
4. Add a name and description for the template
5. Click "Import" to add the template to your library

### Using Templates

1. Browse the template gallery
2. Click "Use Template" on your chosen template
3. The template will be available for use in the Visual Matrix

## Content Generation

### Creating Strategic Motivations

1. Navigate to the Content Generation page
2. Enter a client brief in the Strategic Motivation tab
3. Click "Generate Motivations"
4. Review the generated motivations
5. Click "Approve" on motivations you want to use

### Generating Copy Variations

1. Navigate to the Copy Generation tab
2. Select an approved strategic motivation
3. Configure copy settings:
   - Tone (professional, casual, etc.)
   - Length (short, medium, long)
   - Include CTA (yes/no)
4. Click "Generate Copy"
5. Review the generated copy variations
6. Click "Approve" on variations you want to use

## Visual Matrix

### Creating a Matrix

1. Navigate to the Visual Matrix page
2. Click "Create Matrix"
3. Enter a name and description for the matrix
4. Select platforms (Facebook, Instagram, etc.)
5. Select formats (Video, Image, etc.)
6. Click "Create Matrix"

### Managing Matrix Items

1. Select a matrix from the list
2. Click on a matrix item (intersection of platform and format)
3. Assign assets and templates to the item
4. Add copy from approved variations
5. Click "Save" to update the matrix item

## Approval Workflow

### Creating Approval Requests

1. Navigate to the Approvals page
2. Click "Create Request"
3. Select the content type (matrix item, copy, etc.)
4. Select the specific content to approve
5. Assign the request to a team member
6. Add any additional notes
7. Click "Create Request"

### Responding to Approval Requests

1. Navigate to the Approvals page
2. View all requests in the list
3. Click on a request to view details
4. Add comments if needed
5. Click "Approve," "Reject," or "Request Changes"

## Export Management

### Creating Export Jobs

1. Navigate to the Exports page
2. Click "Create Export"
3. Enter a name and description for the export job
4. Select the content type (matrix item, copy, etc.)
5. Select the specific content to export
6. Choose the export format (MP4, JPG, PNG, GIF, ZIP)
7. Configure export settings (quality, source files)
8. Click "Create Export Job"

### Managing Export Jobs

1. View all export jobs in the list
2. Check the status of each job (pending, processing, completed, failed)
3. Click "Download" on completed jobs to get the exported files
4. Click "Cancel" on pending or processing jobs to stop them

## Account Management

### Updating Your Profile

1. Click your profile icon in the top right corner
2. Select "Profile Settings"
3. Update your information:
   - Name
   - Email
   - Password
   - Profile picture
4. Click "Save Changes"

### Resetting Your Password

1. On the login page, click "Forgot Password"
2. Enter your email address
3. Click "Reset Password"
4. Check your email for a password reset link
5. Follow the link to create a new password

## Troubleshooting

### Common Issues

1. **Can't log in**: Verify your email and password, or use the password reset feature
2. **Content not saving**: Check your internet connection and try again
3. **Export job failed**: Check that all required assets are available and try again
4. **Template import failed**: Verify the Creatomate template ID is correct

### Getting Help

For additional assistance, contact support at support@airwave.example.com or use the help button in the bottom right corner of the platform.

## Best Practices

1. **Organize assets with tags**: Use consistent tagging to make assets easy to find
2. **Create templates for common formats**: Save time by creating reusable templates
3. **Use strategic motivations**: Generate copy based on approved motivations for consistency
4. **Review content in the visual matrix**: See how content looks across different platforms
5. **Get approvals early**: Submit content for approval as soon as it's ready
6. **Export in batches**: Group related content into single export jobs
